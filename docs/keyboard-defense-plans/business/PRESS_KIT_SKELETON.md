# Press Kit Skeleton

Create a press kit folder containing:

## 1) Fact Sheet
- Title
- Genre
- Platforms
- Price (if known)
- Release window
- Contact

## 2) Description
- One-liner
- Short description (50-80 words)
- Long description (200-400 words)

## 3) Features
- Bullet list of key features

## 4) Media
- Logos (transparent PNG + SVG)
- Key art
- Screenshots (6-10)
- Trailer links

## 5) Team
- Short bio
- Credits

## 6) Press Contact
- Email
- Social handles
- Website

See `/marketing/press-kit/` for templates.
