# itch.io Store Copy Template

## Summary
A typing-first strategy roguelite where you command a kingdom by typing.

## How to Play
- Use typed commands like `build`, `explore`, and `repair`.
- You can tune time pressure and error tolerance in Settings.

## What's in this build
- Day/night loop
- Exploration + POIs
- Night defense waves
- Skill report card

## Accessibility
- Adjustable time pressure
- Practice mode
- Reduced motion and high contrast options
