# Privacy Notice Template (Non-Legal)

> This is a template for drafting a privacy notice. Consult a qualified professional for legal advice.

## Summary (Plain Language)
- The game runs offline and does not require an account.
- Telemetry is **off by default**. If you opt in, the game collects limited gameplay metrics to improve balance and typing pedagogy.
- We do not sell personal data.

## Data We May Collect (If Opted In)
- Session length
- Run outcomes (day reached, death cause)
- Aggregate typing stats (WPM, accuracy, error types)
- Settings values (time pressure, tolerance)

## Data We Do Not Collect
- No keystroke logs of arbitrary text outside game prompts (unless explicitly stated and enabled)
- No contact lists, no browsing data

## Storage and Retention
- Data is stored locally first.
- If uploads exist, retention period and deletion process must be described here.

## Contact
Provide an email address for privacy inquiries.
