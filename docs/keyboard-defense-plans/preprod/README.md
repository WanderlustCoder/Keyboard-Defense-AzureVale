# Keyboard Defense (Godot) - Pre-production planning pack

This folder adapts the pre-production planning pack to the current Godot project.
It captures acceptance criteria, pipelines, performance targets, and safeguards
before deeper implementation.

Contents:
- Acceptance criteria and definition of done for Vertical Slice and MVP.
- Wordpack and typing curriculum pipeline.
- Keyboard layout and localization strategy.
- Accessibility and comfort settings.
- Save system spec, telemetry privacy, and migration rules.
- Performance budgets and profiling plan.
- CI/CD and release checklist guidance.
- Risk register and playtest plan.
- Milestone prompts for Codex CLI.
- JSON schemas for savegames and telemetry events.
