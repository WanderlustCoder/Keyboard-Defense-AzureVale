# Keyboard Defense (Godot) - Extended planning pack

This folder adapts the optional extended planning pack to the current Godot
project. It focuses on narrative content systems, exploration events, modding
content packs, balance simulation, and tutorial structure.

Contents:
- Event and POI system specification.
- Narrative content bible and tutorial script.
- Modding and content pack loader specification.
- Monte Carlo balance simulation plan and scenarios.
- Extended implementation order appendix.
- Content seeds (events, POIs, threat cards) for inspiration.
- JSON schemas for events, POIs, and content packs.
- Codex milestone prompts to implement the systems.
- Balancing workbook spec.
