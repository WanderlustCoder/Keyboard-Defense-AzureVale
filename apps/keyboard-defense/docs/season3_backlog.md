# Season 3 Backlog - Mastery Track (Ages 8-16)

Next wave of features for the typing-education castle defense game. Guardrails: ages 8-16, runs in Edge/Chrome, single-player, free (no monetization), cartoonish pixel art for all sprites and castle pieces, no multiplayer yet. Builds on Season 1 and 2 foundations.

## Curriculum and Pedagogy
1. Curriculum tree with finger-mastery nodes and explicit gating.
2. Daily warm-up playlists (5-minute sequences) tuned for accuracy.
3. Rotating posture and wrist micro-tips shown between runs.
4. Bigrams/trigrams auto-suggested micro-drills for missed patterns.
5. Alternating-hand rhythm drills with gentle tempo guidance.
6. Mixed-case drills emphasizing shift timing and hand coordination.
7. Number-row drills that type short math-like expressions.
8. Punctuation sentence practice with gentle pacing prompts.
9. Commonly confused key focus sets (e.g., b/p, g/j) with repetition spacing.
10. Speed plateau detection that recommends focus drills.
11. Accuracy-first advancement gates with soft caps on speed pushes.
12. Long-form passage typing in timed segments with mid-segment breaks.
13. Custom practice builder using the curated safe dictionary only.
14. Rest-and-stretch pacing (Pomodoro-style) with kid-friendly reminders.
15. Regional layout prep module (QWERTZ/AZERTY) behind a toggle.
