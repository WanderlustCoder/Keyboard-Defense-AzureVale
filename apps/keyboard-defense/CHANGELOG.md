# Changelog

All notable changes to Keyboard Defense are recorded here automatically by
`semantic-release`.

> Run `npm run release:dry-run` to preview the next entry before publishing.

## 2025-12-28

- Added a pixel-art biome gallery with HUD card, options shortcut, and overlay to pick an active backdrop, view palettes/tags, and track runs/lessons/drills per biome (stored locally).
- Added a day/night palette selector that swaps global theming for HUD panels/overlays, persists per device, and pairs with biome theming.
- Added layered parallax backgrounds (day/night/storm) with gentle motion that pauses under Reduced Motion/Low Graphics and is selectable in the options overlay.
- Added dynamic music suites with layered stems that react to wave danger/health, plus an options overlay picker with preview.
- Refreshed the SFX library with selectable palettes and an overlay to preview mixes before activating.
- Refreshed the enemy readability guide with silhouette/contrast examples, tier color tags, and an overlay entry point in options.

## 2025-12-27

- Added a personal WPM ladder that records best drill runs per mode, with a HUD card, options shortcut, and overlay that surfaces per-mode standings and overall best runs (stored locally).
- Added a training calendar heatmap that tracks daily lessons/drills, with a HUD card, options shortcut, and overlay grid to visualize recent practice streaks (local-only).
- Added streak-freeze tokens that grant one token after 5-day streaks; HUD card surfaces tokens and streak status with an options shortcut.

## 2025-12-21

- Added lesson medal tiers (bronze/silver/gold/platinum) with HUD panel/overlay, replay CTA from drill summaries, and persisted history per mode.

## 2025-12-22

- Added Mastery Certificate panel/overlay with learner name field, live session stats (lessons, accuracy, WPM, combo, drills, minutes), and download/print button plus options shortcut.

## 2025-12-23

- Added reduced-motion-safe milestone celebration banner that triggers on lesson milestones (5/10/20/30/50/75/100), Gold/Platinum medal wins, and high-accuracy mastery certificates.

## 2025-12-24

- Added adaptive mentor dialogue card that shifts between accuracy, speed, and balanced tips based on live typing accuracy/WPM with cooldowns and reduced-motion-safe styling.

## 2025-12-25

- Added Castle Museum panel and overlay to browse unlocked skins, reward artifacts, companion moods, lore scrolls, medals, certificates, and practice archives with options/HUD entry points.

## 2025-12-26

- Added Side Quest HUD panel and overlay that lists narrative quests (lessons completed, Gold/Platinum medals, lore scrolls, practice drills) with progress bars, status pills, and entry points from both HUD and options.
