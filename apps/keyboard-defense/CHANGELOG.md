# Changelog

All notable changes to Keyboard Defense are recorded here automatically by
`semantic-release`.

> Run `npm run release:dry-run` to preview the next entry before publishing.

## 2025-12-21

- Added lesson medal tiers (bronze/silver/gold/platinum) with HUD panel/overlay, replay CTA from drill summaries, and persisted history per mode.

## 2025-12-22

- Added Mastery Certificate panel/overlay with learner name field, live session stats (lessons, accuracy, WPM, combo, drills, minutes), and download/print button plus options shortcut.

## 2025-12-23

- Added reduced-motion-safe milestone celebration banner that triggers on lesson milestones (5/10/20/30/50/75/100), Gold/Platinum medal wins, and high-accuracy mastery certificates.

## 2025-12-24

- Added adaptive mentor dialogue card that shifts between accuracy, speed, and balanced tips based on live typing accuracy/WPM with cooldowns and reduced-motion-safe styling.

## 2025-12-25

- Added Castle Museum panel and overlay to browse unlocked skins, reward artifacts, companion moods, lore scrolls, medals, certificates, and practice archives with options/HUD entry points.

## 2025-12-26

- Added Side Quest HUD panel and overlay that lists narrative quests (lessons completed, Gold/Platinum medals, lore scrolls, practice drills) with progress bars, status pills, and entry points from both HUD and options.
