# Project Labels and Triage

## Recommended GitHub Labels
Core:
- `bug`
- `enhancement`
- `balance`
- `content`
- `playtest`
- `triage`

Priority:
- `P0`, `P1`, `P2`, `P3`

Area:
- `area:typing`
- `area:sim`
- `area:economy`
- `area:exploration`
- `area:defense`
- `area:save`
- `area:art`
- `area:audio`
- `area:tooling`

Status:
- `blocked`
- `needs-design`
- `needs-test`
- `needs-playtest`

## Triage Rules
- P0 bugs: crashes, save corruption, softlocks
- P1: severe gameplay regression, input failures
- P2: balance issues, confusing UX
- P3: polish, nice-to-have
