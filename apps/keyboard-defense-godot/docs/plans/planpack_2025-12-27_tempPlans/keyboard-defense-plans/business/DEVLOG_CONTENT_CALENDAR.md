# Devlog / Content Calendar

A lightweight plan for communicating progress without overcommitting.

## Cadence Options
- Weekly: small update (screenshot + one feature)
- Biweekly: deeper update (system + clip + roadmap)
- Monthly: "major update" style

## Recommended 6-Week Cycle (example)
Week 1: Typing UX improvements (gif + settings)
Week 2: New enemy type + threat card
Week 3: Exploration POI highlight
Week 4: Art/audio pipeline progress
Week 5: Balance patch notes + playtest call
Week 6: Roadmap update + wishlist CTA

See `marketing/social/ANNOUNCEMENT_TEMPLATES.md`.
