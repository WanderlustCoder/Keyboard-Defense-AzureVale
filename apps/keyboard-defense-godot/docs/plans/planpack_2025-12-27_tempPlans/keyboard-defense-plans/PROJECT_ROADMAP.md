# Project roadmap (Keyboard Defense)

## Phase 0 - Foundation (done)
- Campaign map, battle loop, rewards, and upgrades.
- Headless tests for core systems and layout.

## Phase 1 - UI refit (current)
- SFK-inspired visual direction with original layout.
- Improved HUD readability and hierarchy.
- Keyboard navigation passes and accessibility toggles.

## Phase 2 - Learning depth
- Adaptive typing prompts using error tracking.
- Optional challenge drills with bonus rewards.
- Expanded lesson packs and themed word sets.

## Phase 3 - Content expansion
- Additional campaign nodes and battle variants.
- Enemy and lane identity variations.
- More upgrade paths with distinct play styles.

## Phase 4 - Polish
- Audio, VFX, and animation polish.
- Localization and accessibility presets.
- Performance optimization for low-end machines.

## Risks
- Overwhelming new typists with time pressure.
- UI complexity eroding clarity.
- Balance skew if rewards scale too steeply.
